# TallerGitHub

Práctica guiada para aprender y experimentar con **GitHub Copilot** en un proyecto multi-tecnología.

## Descripción

Este repositorio está diseñado como un taller práctico para explorar el uso de GitHub Copilot en diferentes lenguajes y tipos de archivo:

- SQL
- RPGLE
- SQLRPGLE
- Java
- Python
- Documentación en Markdown

La idea es recorrer ejercicios y ejemplos donde Copilot ayude en generación de código, refactorización, documentación, pruebas y revisión.

## Objetivos del taller

- Entender cómo redactar buenos prompts para Copilot.
- Practicar generación asistida de código en varios lenguajes.
- Documentar decisiones técnicas con Markdown.
- Comparar sugerencias automáticas y validar su calidad.
- Aplicar buenas prácticas de control de versiones en GitHub.

## Alcance tecnológico

Este taller incluirá ejemplos de:

1. Consultas y scripts SQL.
2. Programas y rutinas en RPGLE.
3. Programas embebidos SQLRPGLE.
4. Clases y utilidades en Java.
5. Scripts y automatizaciones en Python.
6. Guías paso a paso y notas en Markdown.

## Requisitos recomendados

- Cuenta de GitHub.
- Visual Studio Code.
- Extensiones:
  - GitHub Copilot
  - GitHub Copilot Chat
- Entorno de ejecución según lenguaje (Java, Python, etc.).

## Flujo de práctica sugerido

1. Elegir un lenguaje o módulo.
2. Definir una tarea pequeña y concreta.
3. Redactar prompt claro para Copilot.
4. Revisar sugerencias (lógica, estilo y seguridad).
5. Probar, ajustar y documentar en Markdown.
6. Guardar cambios con commits atómicos.

## Buenas prácticas durante el taller

- No aceptar sugerencias sin validarlas.
- Pedir explicaciones de código a Copilot cuando sea necesario.
- Mantener prompts y resultados documentados.
- Preferir cambios pequeños y verificables.
- Registrar lecciones aprendidas por cada ejercicio.
